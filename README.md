# AI Intrusion Detection System

## 🚀 Features
- Person Detection using YOLOv8
- Zone-based Intrusion Detection
- Entry-based Alert System
- Suspicious Object Detection

## 🛠 Tech Stack
- Python
- OpenCV
- YOLOv8 (Ultralytics)

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run project:
python main.py

## 📌 Project Description
This project detects intrusion in a restricted area using computer vision. Alerts are triggered only when a person enters a defined zone, reducing false positives.

## 📷 Output
- Real-time detection
- Screenshot capture on intrusion
